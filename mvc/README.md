**Folder structure**

```
    mvc
    |-- lib
    |   |-- main.dart
    |   |-- mine
    |   |   |-- control.dart
    |   |   |-- model.dart
    |   |   |-- view.dart
    |   |-- user_detail
    |   |   |-- control.dart
    |   |   |-- model.dart
    |   |   |-- view.dart
    |   |-- user_list
    |       |-- control.dart
    |       |-- model.dart
    |       |-- view.dart
    |-- test
        |-- mine
        |   |-- model_test.dart
        |   |-- model_test.mocks.dart
        |   |-- view_test.dart
        |   |-- view_test.mocks.dart
        |-- user_detail
        |   |-- model_test.dart
        |   |-- model_test.mocks.dart
        |   |-- view_test.dart
        |   |-- view_test.mocks.dart
        |-- user_list
            |-- model_test.dart
            |-- model_test.mocks.dart
            |-- view_test.dart
            |-- view_test.mocks.dart
```
